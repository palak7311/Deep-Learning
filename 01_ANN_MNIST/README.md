# Handwritten Digit Recognition using ANN

## 📌 Project Overview
This project uses an Artificial Neural Network (ANN) to classify handwritten digits (0–9) from the MNIST dataset.

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

## 🧠 Model Architecture
- Flatten Layer
- Dense (128, ReLU)
- Dense (10, Softmax)

## 📊 Results
- Test Accuracy: **97.78%**

## 📁 Files
- `mnist_ann.ipynb` - Training notebook
- `mnist_ann.keras` - Trained model
